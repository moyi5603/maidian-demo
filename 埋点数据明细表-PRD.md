## 2. 产品概述

### 2.1 产品背景
原DataEase埋点数据明细表存在信息展示冗余、筛选分散、数据维度不够清晰等问题。本优化版本旨在提供更清晰、更高效的埋点数据查看和分析体验。

### 2.2 产品目标
- 提供元素维度的数据聚合视图，方便评估各元素的曝光点击效果
- 优化筛选交互，减少重复和混乱
- 提供清晰的事件明细数据展示
- 支持JSON数据的便捷查看

### 2.3 核心功能
1. **元素数据看板** - 按单个元素维度展示曝光、点击、CTR等核心指标
2. **事件明细表** - 展示详细的埋点事件记录
3. **全局筛选** - 统一的页面级筛选区域
4. **JSON数据查看** - 弹窗形式展示详细JSON数据

## 4. 功能需求

### 4.1 全局筛选栏

#### 4.1.1 筛选项配置

| 筛选项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 租户 | 可搜索下拉框 | 全部租户 | 支持输入搜索，显示匹配的租户选项 |
| 组件名称 | 下拉选择 | 全部组件 | 轮播图、搜索按钮、导航展开、导航收起、热词 |
| 上报时间 | 日期范围 | 近7天 | 开始日期和结束日期两个输入框 |
| 用户ID | 文本输入 | 空 | 精确匹配特定用户 |

#### 4.1.2 租户可搜索下拉框

**交互逻辑：**
1. 点击输入框显示所有租户下拉列表
2. 输入文字时实时过滤匹配的租户（支持按租户名称或ID搜索）
3. 点击选项选中该租户，输入框显示选中的租户名称
4. 点击页面其他区域关闭下拉列表

**租户列表数据：**
- 全部租户
- 租户 1510、1073、1511、1001、1002、1003、2001、2002、2003

#### 4.1.3 操作按钮

| 按钮 | 功能 | 交互 |
|------|------|------|
| 重置 | 清空所有筛选条件 | 点击后恢复默认值，显示提示"筛选条件已重置" |
| 查询 | 执行筛选查询 | 点击后显示提示"查询完成，共找到 X 条记录" |

---

### 4.2 元素数据看板

#### 4.2.1 汇总卡片

展示当前筛选条件下的整体数据概览：

| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 曝光元素数 | 有曝光数据的元素总数 | 去重计数 |
| 总曝光次数 | 所有元素的曝光次数总和 | SUM(expose) |
| 总点击次数 | 所有元素的点击次数总和 | SUM(click) |
| 整体点击率 | 整体CTR | 总点击/总曝光 × 100% |
| 曝光用户数 | 曝光过的独立用户数 | 去重计数 |

#### 4.2.2 元素明细表

**表格列定义：**

| 列名 | 说明 | 排序 |
|------|------|------|
| 元素名称 | 元素的显示名称，如"轮播图-01" | 支持 |
| 元素地址 | 元素的URL或标识 | 支持 |
| 曝光次数 | 该元素的总曝光次数 | 支持 |
| 点击次数 | 该元素的总点击次数 | 支持 |
| 点击率 | CTR，带可视化进度条 | 支持 |
| 曝光人数 | 该元素的独立曝光用户数 | 支持 |

**交互说明：**
- 点击列头可进行升序/降序排序
- 行hover时背景高亮
- 无点击事件（纯展示）

**点击率可视化：**
- 显示为进度条 + 百分比数值
- 进度条颜色：渐变从主色到绿色
- 最大宽度对应100%

#### 4.2.3 分页

| 配置项 | 默认值 | 选项 |
|--------|--------|------|
| 每页条数 | 20 | 10、20、50 |
| 分页控件 | 上一页/下一页 + 页码信息 | - |

### 4.3 事件明细表

#### 4.3.1 表格列定义

| 列名 | 说明 | 示例 |
|------|------|------|
| # | 序号 | 1、2、3... |
| 上报时间 | 事件上报的时间戳 | 2026-05-15 14:23:01 |
| 用户ID | 触发事件的用户标识 | 151277418663 |
| 租户ID | 所属租户 | 1510 |
| 事件名称 | 埋点事件名称 | 新搜索导航搜索曝光 |
| 组件名称 | 所属组件 | 新搜索导航 |
| 设备 | 设备类型 | iPhone、Android |
| 系统 | 操作系统 | iOS 16、Android 12 |
| 网络 | 网络类型 | WiFi、4G、5G |
| 元素地址 | 元素的URL或路径 | https://saas.joybenefits.cn/... |
| 详细数据 | JSON数据查看入口 | ▶ 查看JSON |

**样式说明：**
- 用户ID：纯文本，无标签样式
- 元素地址：灰色文本，不可点击
- 详细数据：蓝色可点击文字

#### 4.3.2 JSON数据弹窗

**触发方式：** 点击"▶ 查看JSON"

**弹窗内容：**
- 标题：📄 详细数据 (JSON)
- 内容：语法高亮的JSON代码
- 关闭：右上角×按钮或点击遮罩层

**JSON语法高亮：**
- Key：蓝色 (#89B4FA)
- String：绿色 (#A6E3A1)
- Number：橙色 (#FAB387)
- 背景：深色 (#1E1E2E)

#### 4.3.3 分页

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 每页条数 | 20 | 可选20、50、100 |
| 页码跳转 | 支持 | 输入页码后点击Go跳转 |
| 总记录数 | 显示 | 格式：显示第 X-Y 条，共 Z 条 |
